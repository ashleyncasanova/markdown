## **Markdown Tutorial**

1. Italics and Bold

    * _Italics_
    ```
        _Italics_
    ```

    * **Bold**
    ```
        **Bold**
    ```

    *   **_Both_**
    ```
        **_Both_**
    ```

1. Headers

    * # Header one
    ```
        # Header one
    ```

    * ## Header two
    ```
        ## Header two
    ```

    * ### Header three
    ```
        ### Header three
    ```

    * #### Header four
    ```
        #### Header four
    ```

    * ##### Header five
    ```
        ##### Header five
    ```

    * ###### Header six
    ```
        ###### Header six
    ```

1. Links

    * Inline links:
        * [Visit GitHub!](www.github.com)
         ```
            [Visit GitHub!](www.github.com)
         ```
        * [You're **really, really** going to want to see this.](www.dailykitten.com)
         ```
            [You're **really, really** going to want to see this.](www.dailykitten.com)
         ```
        * The Latest News from [the BBC](www.bbc.com/news)
         ```
            The Latest News from [the BBC](www.bbc.com/news)
         ```
